# 课程笔记-行测-判断推理 2021-9-19 21:48:40

# 图形推理

  1. # 我的第一感觉-找规律 找到一个可以从选项里选出来答案的理由，那大概率就是对了

  2. 做题提速 要有这个意识 看到什么样的图形就往什么样的方向思考 选项能够两两分类，就分类然后进行两次二选一的判断

  3. # 零散的技巧

  • 元素组成相同 看位置规律

    • 回字走法 内外圈分开看 步数递增或者翻倍之类

  • 元素组成相似 看样式规律 加减同异

    • 注意区分细节点，相减也是一种求异

  • 元素组成不相同，优先考虑属性，再考虑数量

    • 属性有 轴对称或者中心对称性（连续考了12年） 开闭性 曲线直线 习惯：发现轴对称图形立刻画轴(不管是整体对不对称)、等边三角形有三条轴的特点

    • 轴相关的考法，有一些会考轴与图形的关系，比如和图形交点有几个，甚至和几条线相交。 或者单独考轴的方向位置等 或者把图形分开画轴，看多条轴是平行或者垂直的关系。

    • 分割区域相同，黑块数量不同 考虑黑白运算 (也许会和旋转复合考) （黑白运算可以找一大块相同元素一起运算，然后可以秒选答案那种）

    • 切圆，交圆变形、多端点 考虑笔画数 出现日字/田字/五角星变形就可能是考一笔画 当然 一笔画和二笔画可以复合考

      数笔画的快速方法：通过眼睛去除闭合外框 剩下的图形是几笔画就是几笔画

    • 封闭空间被分割 数有几个面 面还可以细分成几边形之类的

    • 出现多边形 线数量

    • 面和框或者交点可能出现复合考法

    • 曲直性：特点：题目出现明显的曲线和直线的图形， 曲线图形、直线图形、曲+直线图形，会考内外（内曲外直，外曲内直）

    • 封闭性：特点：出现有点抽象的图（笑脸飞机糖葫芦等这 种简笔画） 全封闭图形，全开放图形，半开放半封闭图形。

特殊规律：

  • 特点：**两个或者多个封闭图形连在一起。**俩图形之间的关系：点相交，线相交，面相交，相切。

  • 特点：每幅图都出现相同的一个小元素（小箭头，一个点，一个圆，一个弧等等） 起到标记作用。然后就判断标记的点/线/面/角 等等有什么特点

  • 数量规律：元素组成不相同不相似，属性无规律，数量规律明显

    • 面：白色的封闭区域

      • 图形被分割、封闭区域明显
      • 抽象的粗线条留白区域
      • 更细化的考点：所有面的形状、相同形状面的个数、最大面的形状

# 类比推理

  • 语义关系(对比择优) 2-3题

    • 近义反义 选不出来的话更深一层考虑二级辨析褒义贬义中性之类的感情色彩
    • 比喻 象征

  • 逻辑关系

    • 完全相同 ：就是一个东西

    • 并列关系 AB完全不同但是处于同一层级 选不出来的话考虑二级辨析 人工/自然

      • 矛盾关系(二选一) 生/死 开/关 男/女 对称/不对称

      • 反对关系(有第三者) 红色/蓝色 草莓/香蕉

    • 包容关系

      • 种属关系 X是X的某一种
      • 组成关系 X是X的一部分
      • 用X是X来造句辨析 比如镜片是眼镜 不合理 所以是组成关系 白菜是蔬菜 合理 所以是组成关系

    • 交叉关系 题干按不同标准划分一类事物

      • 造句辨析 有的A是B 有的B是A 有的A不是B 有的B不是A 都要通顺

    • 对应关系

      • 材料 X是X的材料之一
      • 功能 的功能是X
      • 属性 X是X的属性
      • 时间顺序 先X再X
      • 因果关系 因为X导致了X
      • W

  • 语法关系

    • 主谓 动宾 主宾

  • 需要积累成语 俗语 词语等
