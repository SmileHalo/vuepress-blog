# 课程笔记-行测-言语理解与表达 2021-9-14 20:36:03

# 逻辑填空（横线处填入哪一句/哪一项最恰当)

  1. 对应关系 (选项填入段落要符合大意)

    1. 要有一定的词语积累(写个成语啥意思你都不知道那还说个屁) 但我发现我不会做的几乎都是积累不够

    2. 放大侧重点（至少选项出现了不同成语有不同的侧重点得懂吧，比如冰山一角、细枝末节、凤毛麟角你要知道一个突出一小部分，一个突出零散细节、一个突出稀少罕有） 我觉得可以把侧重点不同的地方放大来思考选项之间的差异

    3. 根据形象化表达来匹配对应关系

      标志词:有如、犹如、如...一般、好比、就像、或者出现了双引号这类 带有比喻拟人等特征

      eg. (2019国家）尽管有**前车之鉴，但是依然不乏职能部门重蹈覆辙**

      eg. (2020联考）当今，一些篆刻家努力创作出某些颇为**“新鲜”的作品，它们也许是“创新”的前奏，却缺乏艺术土壤的滋养，必会昙花一现而凋谢**。

    4. 段落中的连词

      1. 转折词（而、但、却、不是xx 而是xx)会导致段落意义的对位关系
      2. 连接词 (顿号，逗号、 且、还、不但xx而且xx、或者xx或者xx、既不xx也不xx）会导致段落意义的并列关系
      3. 因果关系 递进关系(之所以xx是因为xx 因为xx所以xx)

    5. 两面词的答案要在文中有两面对应 比如宠辱不惊 舍本逐末 入不敷出 推陈出新 本****末倒置

# 标题填入题(段落中心题) （这段文字意在说明XXX，下列哪一项作为标题最好）

  1. 标题填入其实就是理解段落中心思想，然后找到把中心思想描述得更为准确的选项，如果选项A能够把选项B描述得更准确，体现B选项中不能体现的，那么A描述就更准确
  2. 注意文段结构，总分总，分清主次，以及各选项中不同的用语描述、范围是否恰当
  3. 找准【总】的这个结构： 于是、所以说、可以说、总的来说、那么、综上 等等表示总结的词
  4. 要对文段中的提出的问题，提出的对策进行提炼，文段中若出现（XXX事，要XXX，应当XXX）这类提出对策比较多的文段，那重点就在于总结出来的对策关系

# 接语选择题(这是文章的开头，这篇文章最可能介绍/最不可能介绍____？ )

  1. 重点看尾句的衔接
  2. 这类也要看文章是什么结构，提出了什么问题之类的

# 代词指代题（文中的“这”指的是)

警惕错误选项的描述不明确 片面

# 说法判断 （下列说法错误的是）

仔细读题了就，看理解力

# 句子排序

转折，逻辑顺序，句子的段落（下定义之类的一般考虑为首句，举例子的句子前面应该有一句下定义等等)

# 其他通用的

听课的过程中，发现看老师的题 标注好了关键信息的题我即使不听讲也能准确率很高的选出答案，应该注重如何标记关键信息的这个方面。
